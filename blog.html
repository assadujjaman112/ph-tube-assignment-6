Most Asked Questions

Discuss the scope of var, let, and const

var : The scope of a ' var ' variable is functional scope. ' var ' variable is accessible within the entire function in which it is declared. Though it is declared within a block it is still accessible outside of that block. ' var ' variable can be updated and re-declared into the scope.

let : The scope of a ' let ' variable is block scope. ' let ' variable is only accessible within the block in which it is declared. ' var ' variable can be updated but cannot be re-declared into the scope. ' let ' variable cannot be accessible outside of the block where it is declared.

const : The scope of a ' const ' variable is block scope. ' const ' variable cannot be update or re-declared into the scope. That means once we declare a ' const ' variable we cannot change the value of that variable.

Tell us the use cases of null and undefined

Null : Null indicates the intentional absence of a value.

Undefined : Undefined indicates the unintentional absence of a value.

What do you mean by REST API?

REST API : REST API means - Representational State Transfer Application Programming Interface. REST is a software architectural style that developers apply to web APIs. REST APIs provide simple uniform interfaces because they can be used to make data, content, algorithms, media, and other digital resources available through web URLs.
